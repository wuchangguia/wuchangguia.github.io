    =表示一级标题，-表示二级标题，代码如下

    1. 使用 # 表可以表示1-6级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，依次类推，代码如下： 
        # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题
    ###### 六级标题

    二.Markdown中段落的划分

    Markdown段落的换行是使用两个以上的空格加上回车，或在段落后使用一个空行来表示重新开始一个段落。 

    方法一:a段落_____b段落
方法二:a段落

      b段落

    三.Markdown字体

    Markdown可以使用以下几种字体: 

    *斜体文本*   
_斜体文本_
**粗体文本**
__粗体文本__
***粗斜体文本***
___粗斜体文本___

    四.分隔线

    在一行中使用三个以上的’*’或’-‘或’_’建立一个分割线，这行内不能有其他内容。

    ***
-----
________

    五.删除线

    在需要画删除线的两端加上波浪线~~，此方法有时会被认为是不规范的语法。需要使用del，代码如下：

    ~~内容~~
<del> 我是被删除的 </del>

    效果为：
    删除线 （渲染不规范，’因为前端预览是js渲染的，后端输出php处理的，两者的markdown语法有差异’，引用简书. ）
    我是被删除的内容（使用del)

    七.脚注

    注意 : 1.脚注 id 必须唯一
            2.无论脚注 id 如何起名，显示时一律标为数字，并且按出现顺序排列

    脚注1[^1]

    八.列表

    Markdown 支持有序列表和无序列表。

    无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容：

    * 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项

    效果如下：

    • 第一项
    • 第二项
    • 第三项

    有序列表使用数字并加上 . 号来表示，如：

    1. 第一项  
2. 第二项
3. 第三项

    效果如下：

    1. 第一项
    2. 第二项
    3. 第三项

    列表嵌套只需在子列表中的选项前面添加四个空格即可：

    1. 第一项：
- 第一项嵌套的第一个元素
- 第一项嵌套的第二个元素
2. 第二项：
- 第二项嵌套的第一个元素
- 第二项嵌套的第二个元素

    九.区块

    Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号，
    区块还可以嵌套以及放入列表中，一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推：
    s
    > 区块
    > hi
    > >第二层
    * 列表
    >列表中的区块
    效果如下：

    区块
    hi

    第二层

    十.代码

    代码区块
    如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`），例如： 

     `println();`函数

    效果如下：
    printf()函数

    代码区块使用 4 个空格或者一个制表符（Tab 键）。

    实例如下： 

        public static void main(String args[]){
        System.out.print("Hello World");
}
    • 也可以使用```包裹一段代码。
    • 同时第一行反引号后面，输入代码块所使用的语言，实现代码高亮。
