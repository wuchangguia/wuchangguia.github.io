    通过hexo创建我的博客

    前言

    该博客的搭建学习参考了codesheep的视频是在mac环境下的博客搭建教程,主题是叶落阁的3-hexo，此篇笔记以总结记录Windows10环境下博客搭建的过程以及搭建过程中所遇到的问题。

    软件安装

    • node.js，安装中会附带一个npm软件。
    • git
    • markdownpad2.0 或者 Typora

    ##搭建过程

    下载好所需软件

    在命令行界面利用npm下载hexo博客框架

    • 利用淘宝镜像下载

    npm install -g cnpm --registry=https://registry.npm.taobao.org

    • 等待下载完成后，再输入
      cnpm install -g hexo-cli

    下载完成后即已下载hexo博客框架。可通过hexo -v查看版本信息验证是否安装完成

    新建一个空文件夹

    该文件夹用于存放博客信息，之后命令行的操作都在此文件夹下进行。

    当博客创建失败或想重写博客时可以直接清空该文件夹，重新进行下面的步骤。

    初始化一个博客

    将命令行跳转到之前创建的文件夹下，输入hexo init（该过程需要安装git）。

    可通过hexo s启动博客，进入博客(localhost:4000)查看是否创建成功。在命令行界面键入ctrl+C关闭博客

    将博客部署到github上

    1. 登录github官网

    2. 在github上创建一个仓库，右上角点击加号选择New repository，仓库名字一定要为xxx.github.io xxx为github用户名。

    3. 安装插件在命令行输入cnpm install --save hexo-deployer-git或者npm install hexo-deployer-git --save

    4. 配置hexo，博客根目录下_config.yml ，冒号后一定要加空格

    deploy:
   type: git
   repo: git@github.com:xxx/xxx.github.io.git
   branch: master

    repo为创建仓库后给的地址，直接复制即可。

    1. 部署远端 hexo -d，此时会弹出github界面，要求输入github账号密码。按要求登录即可

    2. 打开 xxx.github.io ，就能看到你的blog了

    选择主题

    1. 可以在hexo官网查看自己喜欢的主题
    2. 通过git clone [url] themes/xxx 将主题克隆到本地
      例如

    git clone https://github.com/litten/hexo-theme-yilia.git themems/yilia

    查看themems目录下是否下载完成yilia文件夹。

    1. 修改之前创建博客所用的根目录下的_config.yml文件 中的theme：xxx

    2. 在博客根目录的命令行界面依次输入以下的命令

    hexo clean
hexo g
hexo d

    博客推送远端完成

    ####常用命令

    #创建一个新的文章
$ hexo n "文章名"

#生成静态文件
$ hexo g

#将一个草稿发布出去
$ hexo publish [layout] <filename>

#启动一个本地服务器
$ hexo s

    更多命令可参考官方文档
